# Microsoft Security Operations Analyst Associate (SC-200)

## Overview
The Microsoft SC-200 (Security Operations Analyst Associate) certification validates the ability to investigate, respond to, and hunt for threats using Microsoft security tools — primarily Microsoft Sentinel, Microsoft Defender XDR (formerly Microsoft 365 Defender), and Microsoft Defender for Cloud. Issued by Microsoft, it is a role-based certification aimed at security operations professionals working in blue team and SOC environments. SC-200 is the entry point into Microsoft's security certification track and is a prerequisite for the SC-100 Cybersecurity Architect Expert certification.

## Who Should Get This
- SOC analysts (Tier 1–3) using Microsoft Sentinel or Defender products
- Security engineers and analysts working in Microsoft Azure or Microsoft 365 environments
- Threat hunters and incident responders at organizations running Microsoft security stack
- IT security professionals transitioning into dedicated SOC roles
- Blue team members looking for a vendor-specific credential in the Microsoft ecosystem
- Those working toward the SC-100 Cybersecurity Architect Expert certification

## Requirements
- No formal prerequisites listed by Microsoft, but practical experience is expected
- Microsoft recommends at least one year of hands-on experience with Microsoft security, compliance, and identity products
- Familiarity with Microsoft Azure fundamentals (AZ-900 level knowledge) is helpful
- Certification requires annual renewal (Microsoft introduced this for role-based certs in 2021)

## Exam Details
- **Exam Code:** SC-200
- **Questions:** 40–60 questions (multiple choice, case studies, drag-and-drop, hot area)
- **Duration:** 120 minutes
- **Passing Score:** 700/1000
- **Fee:** Approximately $165 USD
- **Renewal:** Annual — free renewal assessment via Microsoft Learn (no re-exam fee); must be completed within 6 months of certification expiry notification
- **Testing:** Pearson VUE — in-person or online proctored

## Exam Domains
- Mitigate threats using Microsoft Defender XDR — 25–30%
  - Defender for Endpoint, Defender for Office 365, Defender for Identity, Defender for Cloud Apps
- Mitigate threats using Microsoft Defender for Cloud — 20–25%
  - Security posture management, workload protection, regulatory compliance
- Mitigate threats using Microsoft Sentinel — 50–55%
  - Workspace architecture, data connectors, analytics rules, workbooks
  - Threat hunting using KQL (Kusto Query Language)
  - SOAR with Logic Apps, playbooks, automation rules
  - Incident management and investigation

## Study Tips
- **Timeline:** 6–10 weeks for those with hands-on Microsoft security experience; longer without it
- **Microsoft Learn:** Free official learning paths for SC-200 at learn.microsoft.com — structured, detailed, and directly aligned to exam objectives; this is the primary free resource
- **John Savill's SC-200 Study Cram (YouTube):** Free comprehensive exam prep video — John Savill's channel is the best free resource for all Microsoft certification prep
- **MeasureUp or Whizlabs practice exams:** Official Microsoft practice tests via MeasureUp; Whizlabs is a cheaper alternative with realistic scenario questions
- **KQL (Kusto Query Language) is critical:** Learn how to write queries for hunting and investigation in Sentinel — practice with the SC-200 labs on Microsoft Learn; without KQL comfort, you will struggle with the Sentinel questions
- **Microsoft Sentinel Ninja Training:** Free 4-hour training program from Microsoft's own Sentinel product team — highly practical and covers advanced Sentinel usage
- **Hands-on lab environment:** Use the Microsoft Azure free tier or a trial subscription to deploy Sentinel, connect data connectors, and build analytics rules; you cannot study this cert purely from books
- Microsoft Virtual Training Days (free): Occasionally offered as free live training events aligned to SC-200

## Career Impact
- Qualifies for: SOC Analyst (Microsoft stack), Security Operations Engineer, Threat Hunter (Microsoft), Sentinel Engineer, Cloud Security Analyst, Microsoft Defender Administrator
- Recognized by Microsoft-focused employers, managed security service providers (MSSPs) running Microsoft Sentinel, and enterprises standardized on the Microsoft security ecosystem
- Highly relevant for organizations that have adopted Microsoft 365 E5 or Defender suites — increasingly common in enterprise environments
- Average salary for SC-200-certified professionals: $85,000–$120,000 depending on role, location, and additional certifications
- Required as a prerequisite (along with SC-300 or AZ-500) for the SC-100 Cybersecurity Architect Expert certification

## How It Compares
- **vs. SC-100 (Cybersecurity Architect Expert):** SC-200 is the associate-level analyst credential; SC-100 is the expert-level architect credential requiring SC-200 (or SC-300 or AZ-500) as a prerequisite. SC-200 is the right starting point for SOC and operations roles; SC-100 is the right target for senior architects designing security solutions across the Microsoft stack.
- **vs. GIAC GCIH:** GCIH is vendor-neutral and covers broader incident handling methodology; SC-200 is Microsoft-specific and focuses on using Microsoft's toolset. SC-200 is more applicable if you work in a Microsoft-centric environment; GCIH is more portable across different technologies. For practitioners in Microsoft shops, SC-200 is highly practical; for broader incident response career development, GCIH carries more generalist value.
