# Microsoft Cybersecurity Architect Expert (SC-100)

## Overview
The Microsoft SC-100 (Cybersecurity Architect Expert) is the highest-level Microsoft security certification, validating the ability to design end-to-end cybersecurity solutions across the Microsoft technology stack. Issued by Microsoft, it covers Zero Trust architecture, security operations design, data security, application security, infrastructure security, and governance, risk, and compliance (GRC) across Microsoft Azure, Microsoft 365, and hybrid environments. SC-100 is positioned for senior architects who can translate business risk requirements into security architecture decisions using Microsoft technologies.

## Who Should Get This
- Senior security architects and solution architects designing Microsoft security solutions
- Cloud security engineers moving into architecture roles
- Security leaders responsible for enterprise-wide security design in Microsoft-centric environments
- Experienced professionals who have already earned SC-200, SC-300, or AZ-500 and want to reach the Expert level
- Security consultants and Microsoft partner employees designing solutions for enterprise clients
- CISO and VP-level technical leaders who need to demonstrate architecture competency

## Requirements
- **Formal prerequisite:** Must pass one of the following before earning SC-100:
  - SC-200 (Security Operations Analyst Associate), OR
  - SC-300 (Identity and Access Administrator Associate), OR
  - AZ-500 (Azure Security Engineer Associate)
- Microsoft recommends at least 5 years of security engineering experience, including 3 years with Azure
- Advanced knowledge of Microsoft Azure, Microsoft 365, and security concepts across identity, networking, and data is expected
- Annual renewal required (free renewal assessment via Microsoft Learn)

## Exam Details
- **Exam Code:** SC-100
- **Questions:** 40–60 questions (case studies, multiple choice, drag-and-drop, scenario-based)
- **Duration:** 120 minutes
- **Passing Score:** 700/1000
- **Fee:** Approximately $165 USD
- **Renewal:** Annual — free renewal assessment via Microsoft Learn within 6 months of expiry notification
- **Testing:** Pearson VUE — in-person or online proctored

## Exam Domains
- Design solutions that align with security best practices and priorities — 20–25%
  - Zero Trust strategy, security operations design, resilience
- Design security operations, identity, and compliance capabilities — 25–30%
  - Security operations center design, identity architecture, compliance frameworks
- Design security solutions for infrastructure — 25–30%
  - Hybrid and multi-cloud infrastructure, Defender for Cloud integration, network security
- Design security solutions for applications and data — 20–25%
  - Application threat modeling, DevSecOps, data classification, Microsoft Purview

## Study Tips
- **Timeline:** 2–4 months for experienced security architects already familiar with the Microsoft stack; prerequisite cert experience significantly reduces study time
- **Complete SC-200, SC-300, or AZ-500 first:** The prerequisite cert knowledge is assumed throughout SC-100; without that foundation, the exam is very difficult
- **Microsoft Learn:** Official free learning path for SC-100 at learn.microsoft.com — the primary free preparation resource; read every module in the SC-100 path
- **John Savill's SC-100 Study Cram (YouTube):** Free, comprehensive, highly recommended — Savill is the definitive instructor for Microsoft certification prep
- **Microsoft Cybersecurity Reference Architectures (MCRA):** Download and study the free MCRA whitepaper from Microsoft — this is the architectural foundation behind many SC-100 questions
- **Zero Trust Framework:** Read Microsoft's Zero Trust documentation thoroughly; Zero Trust principles underpin a significant portion of the exam
- **Case study questions:** SC-100 uses complex multi-part case studies — practice reading long scenario descriptions quickly and identifying the key architectural decision being tested
- **Practice exams:** MeasureUp (Microsoft-official partner) for practice tests; scenario-based questions are more representative of the actual exam than simple recall questions

## Career Impact
- Qualifies for: Cybersecurity Architect, Cloud Security Architect, Principal Security Engineer, Security Solutions Architect, CISO (technical leadership track), Microsoft Security Practice Lead
- One of the most prestigious Microsoft-specific security credentials — recognized by Microsoft Partners, large enterprises standardized on Azure, and government agencies using Microsoft cloud
- The Expert-level designation signals senior capability and is used by Microsoft Partners to demonstrate team competency for customer engagements
- Average salary for SC-100-certified professionals: $130,000–$185,000 depending on role, location, company size, and whether combined with broader architecture or CISO-level responsibilities
- Positions holders as senior technical leaders capable of designing enterprise security programs — not just operating tools

## How It Compares
- **vs. CISSP (Certified Information Systems Security Professional):** CISSP is vendor-neutral and the most recognized security management and architecture credential globally. SC-100 is Microsoft-specific and more focused on technical design decisions within the Microsoft ecosystem. For organization-wide, platform-agnostic security leadership, CISSP carries broader recognition. For Microsoft-centric environments, SC-100 is highly relevant and complementary to CISSP. Senior architects often hold both.
- **vs. SC-200 (Security Operations Analyst):** SC-200 is the associate-level operational credential; SC-100 is the expert-level architecture credential. They represent different career paths within Microsoft security — SC-200 for those operating and monitoring security tools, SC-100 for those designing the security architecture. SC-100 requires an associate-level cert (including SC-200) as a prerequisite, making SC-200 a natural stepping stone.
