# GIAC Penetration Tester (GPEN)

## Overview
The GIAC Penetration Tester (GPEN) certification validates the skills required to conduct network penetration tests using best-practice methodologies and techniques. Issued by GIAC, it covers attack techniques across external and internal network environments — including reconnaissance, scanning, exploitation, password attacks, and lateral movement. GPEN is closely aligned with the SANS SEC560 course and is recognized as a strong practitioner-level offensive security credential, particularly for those who prefer a structured methodology approach over purely practical lab-based exams.

## Who Should Get This
- Penetration testers at the junior-to-senior level
- Red team members and offensive security specialists
- Security consultants performing network assessments
- Network and security engineers who need to understand attacker techniques
- Professionals building toward more advanced offensive credentials (OSCP, GXPN)
- Those working in environments where GIAC credentials are specifically recognized or required (e.g., some government and defense contractor roles)

## Requirements
- No formal prerequisites, but a solid networking background (CCNA or Network+ level) and familiarity with Linux and Windows administration are strongly recommended
- Experience with basic scripting (Python, Bash) is helpful
- Certification valid for 4 years; renewable via 36 CPE credits

## Exam Details
- **Questions:** 82–115 questions (varies by exam version)
- **Duration:** 3 hours
- **Format:** Open book (physical printed materials allowed; no electronic devices)
- **Passing Score:** 74%
- **Fee:** Approximately $959 for the exam (includes one retake opportunity)
- **Renewal:** Every 4 years via 36 CPE credits or retaking the exam
- **Testing:** Pearson VUE / ProctorU (in-person or online proctored)

## Exam Domains
- Penetration Testing Methodology and Planning
- Reconnaissance and Open Source Intelligence (OSINT)
- Scanning and Enumeration — Nmap, service fingerprinting, vulnerability scanning
- Exploitation — Metasploit framework, manual exploitation techniques
- Password Attacks — cracking, spraying, pass-the-hash
- Windows Exploitation and Post-Exploitation
- Lateral Movement and Pivoting
- Web Application Attack Basics
- Reporting and Communications

## Study Tips
- **Timeline:** 2–4 months for those with networking and security experience; consider attending SANS SEC560 ("Network Penetration Testing and Ethical Hacking") if budget allows
- **SANS SEC560 course:** The primary aligned course — taught by Ed Skoudis, it is one of the most respected offensive security courses available; attending in-person or via OnDemand provides course books that become your open-book reference
- **Index your notes meticulously:** As with all GIAC exams, the open-book format rewards those with a thorough, well-organized index — build this early and practice using it under timed conditions
- **Metasploit mastery:** Know the framework deeply — search, use, set options, run auxiliary modules, post-exploitation modules; practice in a local lab with Metasploitable or Vulnhub machines
- **TryHackMe / Hack The Box:** Supplement your study with real-world lab environments; GPEN rewards those with genuine hands-on experience even though the exam itself is multiple-choice
- **GIAC practice exams:** Take both included practice tests; they reflect the real exam style and help calibrate your readiness
- **Nmap deeply:** Know scan types (-sS, -sU, -sV, -O, -A), timing flags, output formats, and NSE scripting basics

## Career Impact
- Qualifies for: Penetration Tester, Ethical Hacker, Vulnerability Assessment Analyst, Red Team Operator, Security Consultant (offensive), Security Engineer
- Recognized by government agencies, defense contractors, and major enterprises as a credible offensive security qualification
- Respected within the SANS community and among employers who specifically value GIAC credentials
- Average salary for GPEN-certified professionals: $85,000–$125,000 depending on experience, location, and clearance
- Strong stepping stone toward GXPN (GIAC Exploit Researcher and Advanced Penetration Tester) or OSCP

## How It Compares
- **vs. OSCP (Offensive Security Certified Professional):** OSCP is entirely hands-on — a 24-hour live exam in a lab environment requiring exploitation of multiple machines and a written report. It is widely regarded as the top practical penetration testing credential and is more respected in pure offensive security roles. GPEN is multiple-choice (open book) and easier to achieve, but covers methodology more broadly. GPEN is a credible stepping stone toward OSCP; both can coexist on a resume with OSCP carrying more prestige for offensive roles.
- **vs. CompTIA PenTest+:** PenTest+ covers similar material but also includes engagement management, reporting, and some cloud content. PenTest+ is better for DoD role requirements and broader coverage; GPEN goes deeper on technical attack techniques and carries more weight in the offensive security community. GPEN is generally viewed as more technically rigorous.
