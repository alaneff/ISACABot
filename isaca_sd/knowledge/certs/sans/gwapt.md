# GIAC Web Application Penetration Tester (GWAPT)

## Overview
The GIAC Web Application Penetration Tester (GWAPT) certification validates the skills needed to assess and exploit vulnerabilities in web applications using industry-standard methodologies. Issued by GIAC, it covers the full web application attack lifecycle — from reconnaissance and injection attacks to authentication bypass, session manipulation, and business logic flaws. GWAPT is closely aligned with the SANS SEC542 course ("Web App Penetration Testing and Ethical Hacking") and is respected as a focused, practitioner-level credential for web security specialists.

## Who Should Get This
- Penetration testers who specialize in or want to specialize in web application security
- Application security engineers and developers who need to understand offensive techniques
- Security consultants who conduct web application assessments for clients
- Bug bounty hunters looking to formalize their web security skills
- Red team members who focus on external attack surface including web applications
- Those who hold GPEN and want to add web-specific offensive depth

## Requirements
- No formal prerequisites, but GIAC recommends experience with web technologies (HTTP, HTML, JavaScript, SQL), Linux basics, and familiarity with tools like Burp Suite
- Background in general penetration testing (or GPEN-level knowledge) is strongly recommended before attempting GWAPT
- Certification valid for 4 years; renewable via 36 CPE credits

## Exam Details
- **Questions:** 82–115 questions (varies by exam version)
- **Duration:** 3 hours
- **Format:** Open book (physical printed materials allowed; no electronic devices)
- **Passing Score:** 71%
- **Fee:** Approximately $959 for the exam (includes one retake opportunity)
- **Renewal:** Every 4 years via 36 CPE credits or retaking the exam
- **Testing:** Pearson VUE / ProctorU (in-person or online proctored)

## Exam Domains
- Web Application Fundamentals — HTTP, cookies, sessions, same-origin policy
- Reconnaissance and Application Mapping — crawling, spidering, fingerprinting
- SQL Injection — error-based, blind, time-based, and out-of-band techniques
- Cross-Site Scripting (XSS) — reflected, stored, DOM-based
- Authentication Attacks — credential stuffing, brute force, session fixation, JWT attacks
- Cross-Site Request Forgery (CSRF)
- Business Logic Flaws
- XML/XXE Injection
- Insecure Direct Object References (IDOR)
- File Upload Vulnerabilities
- Burp Suite usage and configuration (extensive)

## Study Tips
- **Timeline:** 2–3 months for those already experienced in penetration testing; longer if web technologies are new to you
- **SANS SEC542 course:** The primary aligned course — highly recommended if budget allows; the course books become your open-book exam reference
- **PortSwigger Web Security Academy:** Free, comprehensive, hands-on web application security labs at portswigger.net — covers every OWASP Top 10 category with practical exercises; this is the best free resource for GWAPT preparation
- **Burp Suite Professional:** Get comfortable with every major feature — Proxy, Repeater, Intruder, Scanner, Comparer, Decoder; practice intercepting and manipulating web traffic in your lab
- **OWASP Top 10:** Know every category cold, understand the exploitation mechanics, and practice finding each type of vulnerability manually (not just with scanners)
- **Build a home lab:** Set up DVWA (Damn Vulnerable Web Application), WebGoat, Juice Shop, and bWAPP locally to practice in a safe environment
- **Index your materials:** Open-book format rewards a well-structured index — organize by attack type, tool, and technique so you can look up anything within seconds
- **GIAC practice exams:** Essential for calibrating readiness; use both included practice tests and pay attention to the explanations for questions you miss

## Career Impact
- Qualifies for: Web Application Penetration Tester, Application Security Analyst, Bug Bounty Hunter, Security Consultant (web focus), AppSec Engineer, Red Team Operator
- Valued by consulting firms that offer web application assessment services, financial institutions, healthcare organizations, and tech companies with significant web surface area
- GWAPT is a strong differentiator for penetration testers who want to show specialized web application expertise alongside a general credential like GPEN or OSCP
- Average salary for GWAPT-certified professionals: $85,000–$130,000 depending on experience, location, and whether it is combined with other credentials
- Excellent complement to GPEN for practitioners who handle both network and web application assessments

## How It Compares
- **vs. GPEN (GIAC Penetration Tester):** GPEN focuses on network penetration testing methodology; GWAPT focuses specifically on web applications. They are complementary, not competing — many professionals hold both. If you do mostly web application assessments, pursue GWAPT first; if you do mostly network assessments, pursue GPEN first. Together they cover the two primary domains of external attack surface.
- **vs. eWPT (eLearnSecurity Web Application Penetration Tester):** eWPT is a practical hands-on exam (report-based, similar to OSCP in format) from INE/eLearnSecurity. It is less recognized in enterprise environments than GWAPT but tests hands-on skills more directly. GWAPT is better recognized in the industry; eWPT is a good alternative for those who prefer practical exams over open-book multiple choice.
